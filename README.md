
**ardwloop** takes care of exchanging a matrix of data between your Java (or Kotlin) program and its matching Arduino
program.

More in the ![Wiki](https://github.com/llschall/ardwloop/wiki) or in the <a href="https://llschall.github.io/ardwloop">ardwloop pages</a>.

Setup:

***build.gradle***

```
repositories {
    mavenCentral()
}

dependencies {
    implementation 'io.github.llschall:ardwloop:0.1.5'
}
```
