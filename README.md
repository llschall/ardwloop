
**ardwloop** takes care of exchanging a matrix of data between your Java (or Kotlin) program and its matching Arduino
program.

Setup:

***build.gradle***

```
repositories {
    mavenCentral()
}

dependencies {
    implementation 'io.github.llschall:ardwloop:0.1.5'
}
```
